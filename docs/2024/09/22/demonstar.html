DemonStar (1997)

The classic vertical shooter w/ multiplayer.

  • wine
  • arcade
  • multiplayer

This post is part of a series featuring old Windows games playable on Linux using Wine. If you are new to Wine, check my Intro To Running Wine Games post for help getting started.

Intro

I played this bad boy in high school. It was pretty cool at the time, bc you could play over the internet w/ a friend.

Install

I don’t think there’s any trick here. I don’t even think there is an installer. You just run the exe file and it works.

Summary

Arch - 32-bit
Runner - sys-wine-9.0
DXVK - dxvk-1.10
VKD3D - proton-2.6
Discrete - ON
Gamescope - OFF
Dependencies - none
Overrides - none

Notes

  • Using Gamescope doesn’t work, the graphics glitch out.
  • The MIDI music player doesn’t work, so no music.
  • Best ingame settings are fullscreen w/ “wide fullscreen” also activated.
  • If you are playing on a laptop like me, you need to know that F2 is START and F3 is PAUSE.
